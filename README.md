# 🚀 Bitcoin Price Forecasting using ARIMA

A **time series forecasting project** that predicts Bitcoin prices using the ARIMA model with statistical analysis.

---

## 🔐 Features

* 📥 **Real-time Data Collection** (Yahoo Finance API)
* 📊 **Time Series Visualization**
* 📉 **Stationarity Check (ADF Test)**
* 🔄 **Differencing for Stationarity**
* 📈 **ACF & PACF Analysis**
* 🤖 **ARIMA Model Implementation**
* 🔮 **Future Price Forecasting**
* 📊 **Actual vs Predicted Visualization**

---

## 🧠 Tech Stack

* 🐍 Python  
* 📊 pandas  
* 📉 matplotlib

## 📊 Workflow

1. Import libraries  
2. Fetch Bitcoin data  
3. Visualize time series  
4. Perform ADF test  
5. Apply differencing  
6. Re-check stationarity  
7. Plot ACF & PACF  
8. Build ARIMA model  
9. Forecast future values  
10. Plot results  

---

## 📊 Model Details

* Model: **ARIMA**  
* Order: **(1,1,1)**  
* Differencing applied for stationarity
